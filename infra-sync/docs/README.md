# InfraSync – DriftGuard Partner Collateral

This folder contains AWS Partner Central–ready assets that describe the InfraSync – DriftGuard for AWS solution.

- `architecture.png` – High-level architecture diagram illustrating event flow, integrations, and optional dashboard.
- `specification-sheet.pdf` – One-page feature summary, technology stack, and deployment footprint.
- `case-study.pdf` – Two-page customer case study covering the challenge, solution, and business outcomes.

Use these assets alongside the repository `README.md`, `SECURITY.md`, and Terraform modules to prepare listings or partner submissions.
